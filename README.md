## Features

- Uses SVG and Vanilla JavaScript with HTML.
- Drag and Drop (TODO).
- Visual Shape property editor.
- Layer System.
- Custom shape creator (TODO).
- Timeline animation creator (TODO).


## Tech Stack

**Client:** Vanilla JavaScript, HTML, CSS3

**Server:** Any HTTP server.
